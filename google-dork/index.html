What is Google Dorking?

Google Dorking (also known as Google Hacking) uses advanced search operators to find information that isn't typically accessible through regular searches, but is publicly available and indexed by search engines like Google.

This exercise demonstrates how easily sensitive or misconfigured information (like directories of files, login pages, or specific file types) can sometimes be found online if websites are not properly secured. It's a powerful tool for both security researchers and, unfortunately, attackers.

Try this: Type in a Google Dork query and click "Search".
Examples:

  • intitle:"index of" confidential
  • filetype:pdf site:gov
  • inurl:login site:edu
